Some of my Projects

Chef Prep:

Web Development in Python with Django

Chef Prep is an application that helps people to find and plan their meals. The project was built with Python using the Django framework. This project was a group project in my Programming Studio class. Through this project I got a chance to work as an Agile Project Manager, so I learned a ton about Agile Development, Web Development, and how to work effectively on a team. I also got experience working with APIs, including Google Authentication. The project is hosted at chefprep.pythonanywhere.com.


Bash Shell:

OS Level Coding in C++

With this project I recreated the Linux terminal with most of its operations. The shell is capable of instruction piping, IO redirection, and supports any built-in Linux command. With this project, I learned an unbelievable amount about all of the different terminal functions that exist in Linux and just how powerful the command line can be. I also got experience writing a parser, working with file descriptors, and using parent-child processes with fork.


Client Server Processes:

Threading, Sockets, and IPC Programming in C++

I completed this project in my Introductory Operating Systems class. The project is a deep dive into the client-server relationship. We continously built on this project, starting from Pipe communications, adding Shared Memory and Message Queues, then adding Concurrency with Threads and Bounded Buffers, and finally utilizing TCP/IP to transfer files over the internet. There are three different GitHub links above, and each one corresponds to a repository on a different transfer protocol.


GPA Calculator:

First Side Project in Java

My GPA Calculator is the very first side project that I ever did. It was also the first time that I actually took time to sit and plan out how everything would work before I actually started writing any code. The project itself is just a simple command line calculator, but it reminds me of how far I've come since I started coding.
